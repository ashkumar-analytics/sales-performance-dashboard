# Executive Sales Performance Dashboard

## Business Problem
Leadership teams need a clear, consistent view of sales performance across
products, regions, and time to support strategic decisions.

## Solution
Designed a SQL-driven executive sales dashboard using a star-schema data model,
with KPIs and trends surfaced through Power BIâ€“ready measures.

## Tech Stack
SQL, Power BI, DAX, CSV-based fact & dimension tables

## Dashboard Structure
- Executive Overview
- Product Performance Analysis
- Regional Performance Analysis

## Note on Data
The dataset is a representative sample inspired by real-world sales schemas.
In production, this approach scales to millions of records using SQL warehouses.

## Key Outcomes
- Clear revenue visibility
- Faster executive decision-making
- Consistent KPI definitions
